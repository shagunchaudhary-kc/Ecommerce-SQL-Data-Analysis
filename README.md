# Ecommerce SQL Data Analysis

# Project Overview
This project performs SQL-based data analysis on an e-commerce dataset to understand sales performance, customer behavior, product contribution, and payment trends.

The analysis focuses on generating business KPIs and insights using structured SQL queries.

# Objectives
- Calculate key business KPIs
- Analyze sales trends over time
- Identify top-performing product categories
- Understand regional sales distribution
- Analyze customer payment preferences

# Tools & Technologies
- SQL (MySQL)
- Dataset: Olist Brazilian E-commerce Dataset

# Key KPIs Calculated
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Monthly Sales Trend

# Insights Generated
- Revenue contribution by product category
- Sales distribution across customer states
- Preferred payment methods
- Monthly sales performance patterns

# Project Structure
- SQL_Queries: Contains all analysis queries
- README.md: Project documentation

# Conclusion
This project demonstrates the use of SQL for real-world e-commerce data analysis and the ability to extract actionable business insights from raw data.
